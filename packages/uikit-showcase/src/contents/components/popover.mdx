
`popover-target` acts as the trigger for the popover.

`popover-content` contains the content that will be shown in the popover itself.
`popover` will wrap `popover-content`.

`popover-wrapper` is present so that both the **popover** and the **target** will be wrapped in a single element.

<span className="hk1 popover-wrapper">
    <span className="hk1 popover-target">
        <Button text="My target" />
    </span>
    <div className="hk1 popover">
        <div className="hk1 popover-content">
        </div>
    </div>
</span>
